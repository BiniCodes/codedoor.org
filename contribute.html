HOW YOU CAN CONTRIBUTE

For Students and Teachers

Register for free, start to learn how to code and join other students and tutors on and through CodeDoor.org.

For Donors

Donate a used operational laptop to the refugees, so they can learn how to code.

For Developers

Join the CodeDoor developer community to code open source modules for and with the refugees and if you want even act as a tutor.
